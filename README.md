# 🌍 Smart AI Travel Planner

Smart AI Travel Planner is a web-based application that helps users plan their trips easily. Instead of searching for everything manually, users just need to enter details like the destination, budget, and number of travel days. The application then generates a personalized travel itinerary using AI.

This project was developed as part of my academic learning to understand how AI can be integrated into web applications and to solve a common real-world problem.

## 🚀 Features

* Generates a personalized travel itinerary based on user input.
* Takes destination, budget, and trip duration into account.
* Uses the Groq API to generate AI-based travel suggestions.
* Simple, responsive, and easy-to-use interface.

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **AI Integration:** Groq API
* **Frontend:** HTML, CSS, JavaScript

## 📁 Project Structure

The project consists of two main folders:

* **Backend** – Contains the Express server, API routes, and environment configuration files.
* **Frontend** – Contains the HTML, CSS, and JavaScript files for the user interface.
